### ম্যাথামেটিকায় ফাংশন বানানোর নিয়মঃ
Syntax:
```




FUNCTIONNAME[INPUTS_]:= FORMULA....




```

---

Example:
```



APlusBSqr[a_, b_] := a^2 + 2*a*b + b^2



APlusBSqr[2, 3]



```
